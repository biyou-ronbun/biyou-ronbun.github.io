# 文体・薬機法ガイド

このファイルは **secretary が管理**します。writer と x-operator は執筆前に必ずここを読みます。
ルールが増えたら、口頭で終わらせずここに追記してください。

---

## 1. 読者

20〜40代の、美容が好きな一般の女性。化粧品は好きだが、論文は読まない。
「難しいことは知りたくない。でも、騙されたくもない」——この2つを同時に叶えるのがこのブログです。

## 2. このブログの立ち位置

**根拠のある美容情報を、煽らずに届ける。**

やらないこと:
- 断定して不安を煽る（「知らないと損をする」「危険な成分」）
- 特定商品のゴリ押し、他社製品への攻撃
- 論文の結果を拡大解釈する

やること:
- 数字をそのまま出す
- 「ここまでは言えない」を必ず書く
- 読者が今夜からできる行動に落とす

## 3. 文体ルール

- です・ます調
- 一文は短く。読点で長く繋がない
- 一人称（「私は」）を多用しない
- 専門用語は必ずその場で通訳する（例: 線維芽細胞（肌のハリのもとを作る細胞））
- 絵文字・装飾記号（★ ♪ など）は使わない
- 「実は」「本当に」は1記事1回まで
- 数字は具体的に（「効果があった」→「12週間で18%改善した」）

## 4. 薬機法 NG表現集

化粧品・健康食品について、以下は**書いた時点で違法**です。

### 絶対NG（医薬品的な効能効果）

| NG | なぜ | 言い換え |
|---|---|---|
| シミが消える / 取れる | 化粧品に治療効果は謳えない | メラニンの生成を抑え、日やけによるシミ・そばかすを防ぐ（※医薬部外品の場合） |
| シワがなくなる / 若返る | 同上 | 乾燥による小じわを目立たなくする（※効能評価試験済み表示がある製品のみ） |
| アンチエイジング | 老化を止める＝医薬品的 | エイジングケア（＝年齢に応じたお手入れ、の意味に限る） |
| 治る / 治療 / 改善する | 医薬品的 | 「〜という研究結果が報告されています」（事実の紹介として書く） |
| 効く | 医薬品的 | 「研究では〇〇という結果でした」 |
| ニキビが治る / アトピーが治る | 疾患への言及 | 疾患名に触れず、肌を清潔に保つ等の範囲に留める |
| 細胞が生まれ変わる / 肌が再生する | 身体の機能変化＝医薬品的 | 使わない |
| デトックス / 毒素を排出 | 根拠なし＋医薬品的 | 使わない |

### 使ってはいけない表現

- 最大級表現: 「No.1」「最高」「絶対」「完全に」「必ず」
- 安全性の断定: 「副作用がない」「100%安全」「無添加だから安心」
- 医師の推奨を装う表現（実際の監修がない限り）
- ビフォーアフター写真での効果の暗示

### 使える表現（化粧品の56の効能効果の範囲）

「うるおいを与える」「肌を整える」「キメを整える」「肌にハリを与える」
「乾燥を防ぐ」「日やけによるシミ・そばかすを防ぐ」「肌をひきしめる」

### 健康食品（サプリ・ドリンク）を扱うとき

化粧品より制限が厳しい。**健康食品に「肌への効果」を謳うことはできません。**

- **「シワ」に触れるときは、試験の測定結果の報告としてのみ書く。** 「その試験で、シワのグレードが有意に改善した（p=0.013）」はOK。ここから読者の肌に接続した瞬間（「あなたのシワにも」「シワが気になる人は飲むといい」）にNGになります。**主語は必ず試験参加者であって、読者ではありません。**
- 水分量・弾力など他の項目も同じ。書けるのは「その試験で何が測られ、どう動いたか」まで。
- 「飲めば〜になる」「〜に効く」は、どんな数字を添えてもNG。
- 摂取をすすめる書き方をしない。「試すなら期間を区切る」のように、**すでに試す人に向けた条件づけ**に留める。

### 迷ったときの原則

**判断がつかない表現は、使わずに済ませる。** 言い換えを探すより、その一文を削ったほうが速く、記事も良くなります。

## 4.5 「日常への生かし方」は毎回必須（オーナー指示・2026-07-11）

**すべての記事に「日常への生かし方」セクションを入れます。例外はありません。** これが無い記事は未完成です。

読者は論文が読みたいのではなく、「で、私は今夜どうすればいいの?」の答えが欲しくて来ます。

- **行動まで下ろす。** 「継続が大切」は心構えであって生かし方ではない。「8週間続けて、変わらなければやめる」まで書く
- **「研究の話はここまで。ここからは提案です」と明示的に宣言してから書く。** 論文の結果と、このブログの提案を地続きにしない
- **効果を約束せず、判断の基準を渡す。** 「飲みましょう」はNG、「効果ありとした試験の用量は1〜10gとばらついています。『◯g飲めば』と断言する広告は、根拠より先を言っています」はOK。読者に渡すのは商品ではなく、**自分で決められる目**
- **生かし方が無いなら「無い」と書く。** 「今のものを変える理由は、この研究からは出てきません」「買い足す必要はありません」も立派な生かし方。無理やり行動を捻り出さない

詳しい書き分けは `.claude/agents/writer.md` の「『日常への生かし方』の書き方」を参照。

## 4.6 PV を取りにいかない（オーナー承認・2026-07-12、同日 改定）

> **【改定の記録】** 初版では「このブログは広告もアフィリエイトも載せません（恒久的な決定）」と書いていました。
> **その後オーナーが方針を変え、広告（Google AdSense）とアフィリエイトを導入する判断をしました。**
> 事実と食い違う宣言を残しておくわけにいかないので、書き換えます。
> 「出典をたどると根拠が無い」を書くブログが、自分の記述を放置するわけにはいきません。

**このブログは広告とアフィリエイトを掲載します。ただし、以下を必ず守ります。**

- **化粧品・サプリメント・美容医療・ダイエットの広告カテゴリはブロックする。**
  検証している業界の広告を、その検証記事の横に出さないため。
- **広告が理由で、記事の中身を変えない。** 「この商品には根拠がありません」は、これまでどおり書きます。
  書けなくなるなら、その収益源を捨てます。
- **アフィリエイトを含む記事には、PR表記を必ず入れる**（景品表示法・ステマ規制。仕組み上、外せません）。
- **商品説明に効能を書かない**（薬機法66条。処罰されるのは広告主ではなく、書いた人です）。

**それでも、PV を目的に書かないという方針は変わりません。**
KPI は **「読者リストの登録数」と「読者からの質問数」** です。PV は補助指標です。

writer と x-operator は、この一行だけ覚えてください。

> **検索流入を狙って薄く書くくらいなら、読者が質問を送りたくなる濃さで書け。**

### やってはいけないこと

- **検索キーワードのために、内容の無い段落を足す。** 文字数を増やす目的の加筆は禁止です。**削るほうが正解**です。
- **「〇〇 おすすめ」「〇〇 ランキング」のような、検索需要だけが動機の構成**にする。
- **見出しにキーワードを詰め込む。** 見出しは検索エンジンではなく、読者が読む位置を探すためのものです。
- **結論を薄めて、記事を分割する。** 1記事で言い切れることを2記事に割らない。
- x-operator: **クリック数だけを狙った煽り**（「知らないと損」「9割が間違えている」）。§2 の「煽らない」に反します。

### 代わりにやること

- **1本の記事で、読者が「これは他では読めなかった」と思う箇所を最低1つ作る。**
  出典をたどった結果、資金提供元、動物実験だった事実——**このブログにしか書けない一段**のことです。
- **答えを全部きれいに閉じない。** 記事の終わりに **「この記事では答えを出せなかったこと」を1つ残す。**
  読者が質問を送りたくなるのは、**問いが残ったとき**です。閉じきった記事からは、質問は来ません。
- **「調べてほしいことはありますか」への導線を、記事末に置く。** これは売り込みではありません。
  読者の疑問を受け取る窓であり、このブログのネタ帳の入口です。
- x-operator: ポストのゴールは **imp ではなく「読みたい」と「聞きたい」**。
  数字が伸びなかったポストより、**1件の質問を連れてきたポストのほうが価値があります。**

### 迷ったときの判定

> **この段落は、読者の判断力を1つ増やしているか?**
> 増やしていないなら、それは PV のために書いた段落です。**消してください。**

## 4.7 「この試験に参加したのは誰か」を必ず1行書く（2026-07-13 追加）

**研究を紹介するとき、その試験に参加したのが誰だったかを、必ず1行で書いてください。**

- 何人か
- 何歳くらいか
- どの国の人か
- 男性か女性か
- 健康な人か、疾患のある人か
- **どこの肌を測ったのか**（顔か、前腕か、手の甲か）

### なぜか

読者は「この研究は、私に当てはまるのか?」を知りたいのに、**それを判断する材料を渡されていません。**

- 白人女性30人の試験を、日本人の読者にそのまま当てはめられるのか
- **前腕**で測ったシワの数字を、**顔**の話として読んでいいのか
- 平均52歳の試験を、25歳の読者が読んでいる

**この1行があるかないかで、読者が自分で判断できるかどうかが決まります。**

これは §4「主語は必ず試験参加者であって、読者ではありません」の実行手順です。**主語が試験参加者だと言うなら、その参加者が誰なのかを書かなければ、読者は照合できません。**

### 書き方

> 903人を4.5年追ったランダム化比較試験です。**参加者はオーストラリアの成人で、評価したのは手の甲。顔ではありません。**

> 女性64人が、1日1g、12週間。**平均年齢は40〜60歳の韓国人女性です。**

**「n=48」とだけ書くのは、書いていないのと同じです。**

## 5. 論文の扱い

- 論文カード（`research/`）にない主張は書かない
- in vitro（培養細胞）や動物実験の結果を、人間の肌の話として書かない
- 「研究では」と書くときは、必ず参考文献に出典を載せる
- 業界資金が入った研究は、その事実を隠さず書く

### 「未確認」の扱い（1本目で確立）

- **論文カードが「未確認」としている項目は、記事でも断定しない。数値も書かない。** 有料で本文が読めなかった効果量、一次情報で確認できなかった資金提供元などが該当します。「たぶんこうだろう」で埋めない。
- 確認できなかったこと自体を、記事に書いてよい（むしろ書く）。「この数値は本文が有料のため確認できていません。だから書きません」は、このブログの信頼そのものです。
- 逆に、**根拠が弱いという事実を根拠に、逆方向に断定するのも禁止**。「ヒトの皮膚で確認したデータは見つからなかった」は書けますが、「肌には届かない」は書けません。
- COI（利益相反）は申告文だけで判断しない。**著者の所属まで見る。** 「利益相反なし」と書かれた論文の著者が製品メーカー所属だった、という事例が実際にありました。

### 安全性について

- **1つの試験を根拠に、安全性を書かない。** 「副作用は報告されなかった」と書ける試験があっても、それは数十人・数ヶ月の話です。安全性の結論には使えません。
- 「副作用がない」「安全です」は、複数試験があっても書かない（§4の断定NGと同じ）。
- 代わりに書くのは、試験の規模と期間の限界（「参加者は64〜99人、期間は8〜12週間の小規模・短期です」）と、医師・薬剤師への相談の一文。
