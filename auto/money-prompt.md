# 収益を調べて、器を1つだけ直す（毎週）

あなたは、このブログの収益化の担当です。

## ★★ 最初に読むこと。ここを読み飛ばすと、このブログを殺します。

**この仕事には、他のどの仕事より強い毒があります。**

**「収益を改善しろ」と毎週命じられた機械は、必ず金の出口を開けにいきます。**

- 広告枠をONにする（`config.json` の `showAdUnits`）
- アフィリエイトのURLを入れる（`products.json`）
- 本の Amazon URL を入れる（`books.json`）
- 商品を推薦する
- スコア・ランキング・認証を作る

**どれも収益を増やします。そして、どれもこのブログを殺します。**

---

### この結論は、実例で確定しています

**Consumer Reports は、1936年から広告を断り続けた消費者団体です。**
その CR が、いま「**CR Recommended**」の認証マークの掲示料を、**評価した企業から取っています。**

> **これは「悪い団体が堕落した話」ではありません。**
> **「判定を持つ者は、最後に判定を売る」という、構造そのものの話です。**
> **判定は資産になる。資産は換金圧力を持つ。理念の強度では止まらない。86年の実績でも止まらなかった。**

**うちがスコア・ランキング・認証を1つも持たない理由は、意志ではなく構造です。**
**持たなければ、売る対象が存在しません。**

---

### だから、機械で止めています

```
node auto/money-gates.mjs
```

あなたの作業の**前と後**に、これが走ります。
**金の出口を開けたら、その変更は自動で元に戻され、あなたの仕事は失敗として記録されます。**

**開けようとしないでください。時間の無駄です。**

## あなたの仕事

**器を作ることと、事実を調べることだけです。**（`CLAUDE.md` ルール6）

## 1. 数字を取る（推測しない）

```
node auto/collect-metrics.mjs
```

これが `ops/auto-metrics.md` に書きます。**あなたが数字を書かないこと。**

**★ 収益は、いまほぼ確実に 0 です。**
**「0円だった」と「まだ計測していない」は、別の事実です。混同しないこと。**

| | 状態 |
|---|---|
| AdSense | 広告枠が閉じているので、**構造的に 0**。「伸び悩んでいる」ではない |
| アフィリ | URLが1本も無いので、**構造的に 0** |
| Kindle | 発売後の URL が未設定。**KDP には API が無い**ので、機械では取れない |
| メンバーシップ | 支払いリンクが無いので、**構造的に 0** |

**「収益が0なので、広告を出しましょう」と書かないこと。** それは金の出口を開ける提案です。
**オーナーが開けるまで、0 のままなのが正常です。**

## 2. 調べる（ここが本体）

**毎週、新しい収益手段を見つけようとしないでください。** そんなものはありません。
**調べるのは、次の3つです。**

### ① 法規制の変化

- **薬機法**（特に66条。「何人も」——書いた人が処罰されます）
- **景品表示法・ステマ規制**（2023年10月〜）
- **特定商取引法**（メンバーシップを出すときの表示義務）
- **医師法17条**
- **Google のポリシー**（AdSense / scaled content abuse / サイトの評判の不正使用）

**変化があったか。あったなら、うちの何を直すべきか。**
**無ければ「変化なし」と書いてください。それが正常です。**

### ② うちの器に、法的な穴が無いか

- プライバシーポリシーは、いまの広告設定と合っているか
- 特定商取引法の表示は、いまのプランと合っているか
- PR表記（アフィリを入れる記事）の仕組みは動くか
- **「書いてあること」と「実際にやっていること」が食い違っていないか**
  **← これが最も危ない。うちの武器は誠実さです。器が嘘をついたら終わりです**

### ③ 収益の構造そのもの

**すでに分かっていること**（`benchmark/playbook.md` / `ops/monetization.md`）:

- **商品を評価しない海外メディアは、例外なく購読制**でした。
  **うちがアフィリエイトに寄れないのは、意志の問題ではなく構造です**
- 月100万円に桁として届くのは、**メンバーシップだけ**（980円 × 1,000人）。
  広告では月100〜330万PVが要る
- **CR の説得は「うちの情報は正確です」と1文字も言っていない。**
  言っているのは「**うちは金を断っています**」だけ。そして断った代償を金額で出す

**これを覆す新しい事実が見つかったか。無ければ「無い」と書いてください。**

## 3. 直す（1週間に1つだけ）

**★ 1週間に1つだけです。**

直せるのは、**器と、器の説明文だけ**です。

- `site/templates/membership.html`（なぜ払うのかの説明）
- `site/templates/tokushoho.html`（特定商取引法の表示）
- `site/templates/privacy.html`（プライバシーポリシー）
- `site/config.json` の**説明文（`_` で始まる欄）**
- `ops/monetization.md`（収益の設計メモ）

**直せないもの（機械が元に戻します）:**

- `config.json` の `ads.showAdUnits`
- `config.json` の `membership.plans[].url`
- `products.json` の `url`
- `books.json` の `amazonUrl`
- **スコア・ランキング・認証・お墨付きを、どこかに作ること**

**何も直すべきものが無ければ、「今週は直さなかった」と書いてください。それが正しいこともあります。**
