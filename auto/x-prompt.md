## 3. 書き方

### ★ X が何を評価しているか（X 自身が公開しているアルゴリズムの重み）

| 行動 | 重み |
|---|---|
| **返信** | **13.5** |
| いいね | 0.5 |
| **自分の投稿への返信に、自分が返す** | **75.0** |
| プロフィールを見て反応 | 12.0 |
| **画像への加点** | **なし** |
| ブロック | **−74.0** |
| 報告 | **−369.0** |

**返信は、いいねの27倍です。**

そして——**うちは今、X が最も低く評価する行動（一方通行の投稿）だけを、1日6回やっています。**

### だから、こう書いてください

**① 返信されうる形で終わること。**
「質問はありますか?」と乞わないこと（乞うても返信は来ません）。
**代わりに、事実の穴を1つ残して終わる。**「この試験は12人でした。あなたの肌で同じことが起きるかは、分かりません。」

**② 「怒り」ではなく「驚き」で書く。**
1行目に「**誰が悪いか**」を書かないこと。**主語を業界ではなく、出典にすること。**

```
✗  美容業界はずっとこの嘘をついてきました
○  「28日周期」の出典を辿りました。実測した論文は、見つかりませんでした
```

**③ 煽らない理由は、道徳ではなく算数です。**
煽りが伸びるのは本当です。**しかし、煽りはいいね（+0.5）とブロック（−74.0）を同時に生みます。**
**大きいアカウントは煽っても生き残る。フォロワー0のうちは、1回で沈みます。**

**④ 画像を無理に増やさないこと。**
X の重み表に、**画像への加点は1つもありません。** 「画像があると伸びる」という根拠は、
うちが確認した範囲では見つかりませんでした。**根拠なく比率を動かさないこと。**

**⑤ スレッドの1本目で、結論を出し切ること。** 焦らさない。引っ張らない。

### ★★ 読者を主語にしないこと（機械が止めます）

**「あなたは騙されていました」は、禁止語を1語も使わずに書けます。**

うちの投稿は「あなたが信じていたことには根拠がなかった」という構造を持っています。
**一歩間違えると「騙されてましたね」になります。そして読者は去ります。**

**読者は敵ではありません。同じものを調べている仲間です。**

```
✗  あなたが信じていた「28日周期」に、根拠はありませんでした
✗  9割が知らない話です
○  「28日周期」の出典を辿りました。実測した論文は、見つかりませんでした
```

**読者を主語にしない。出典を主語にする。**
（`auto/verify-x.mjs` が、この形を機械で止めます）

`.claude/agents/x-operator.md` に従います。加えて **X 固有の決まり**：

- **1行目で終わらせない。** 「〜でした」で始めて、続きを読ませる
- **数字を出す。** 「参加者12人」「著者7人中5人がメーカー社員」。数字は具体的なほど強い
- **煽らない。** 「騙されてる」「嘘」「詐欺」——**これを書いた瞬間、検査が投稿を止めます**
- **「効かない」と書かない。** 書けるのは「出典が見つからなかった」まで
- **絵文字を使わない。ハッシュタグは使わない。** このアカウントの声ではありません
- **URLは、記事がある場合だけ。** 無理に貼らない

### スレッド（連投）を、週に2本は入れてください

単発より読まれます。5ポスト前後。`replyToLocalId` で親を指します。
**最後のポストには、必ず「ここまでは言えません」を入れてください。** 伸びる連投ほど、断定に寄る力が働きます。

## 5. ★ 検査を通す（ここが関門）

```
node auto/verify-x.mjs
```

- **通ったら** → 6へ
- **落ちたら** → 落ちた投稿を直して、通るまで繰り返す。**直せなければ、その投稿を削除する。**
  **落ちた投稿を無理やり通すために、検査のほうを緩めないこと。**

検査が止めるもの: 薬機法の禁止表現 / 煽り（嘘・詐欺・騙し）/ 言い過ぎ（効かない・やめるべき）/ 文字数超過。

**なお「『効かない』とは書けません」のように、言い過ぎを断っている文は通ります。**
関門は「言い切っているか」を見ています。
