### ① `site/claims.json`（51件・最大の在庫）

読者が信じている言説と、その出典を辿った結果が入っています。**1件が、そのまま1投稿になります。**

- `question` … 読者の言葉（これが投稿の1行目になる）
- `claim` … 世間で言われていること
- `traced` … 出典を辿った結果
- `found` / `note` … 実際に見つかったもの
- `pmids` … 論文

`traced` の値ごとに、書ける最大のことが変わります。**ここを間違えると、このブログは終わります。**

| traced | 書けること | **書けないこと** |
|---|---|---|
| `no-source` | 「その主張の出典が見つかりませんでした」 | **「効かない」「嘘」** |
| `animal-only` | 「確かめられたのはラット（またはブタ）でした」 | 「人間にも同じことが起きる」 |
| `invitro-only` | 「培養細胞での結果です」 | 「あなたの肌でも同じ」 |
| `lab-measure-only` | 「機械での測定値です。見た目は測っていません」 | 「見た目が変わる」 |
| `industry-only` | 「独立資金の試験は見つかりませんでした」 | **「嘘」「捏造」** |
| `human-trial` | 参加者の人数・年齢・国・測った部位を**必ず添えて**結果を書く | 「n=48」だけ書いて終わること |

## 3. 書き方

`.claude/agents/x-operator.md` に従います。加えて **X 固有の決まり**：

- **1行目で終わらせない。** 「〜でした」で始めて、続きを読ませる
- **数字を出す。** 「参加者12人」「著者7人中5人がメーカー社員」。数字は具体的なほど強い
- **煽らない。** 「騙されてる」「嘘」「詐欺」——**これを書いた瞬間、検査が投稿を止めます**
- **「効かない」と書かない。** 書けるのは「出典が見つからなかった」まで
- **絵文字を使わない。ハッシュタグは使わない。** このアカウントの声ではありません
- **URLは、記事がある場合だけ。** 無理に貼らない

### スレッド（連投）を、週に2本は入れてください

単発より読まれます。5ポスト前後。`replyToLocalId` で親を指します。
**最後のポストには、必ず「ここまでは言えません」を入れてください。** 伸びる連投ほど、断定に寄る力が働きます。

```json

  "replyToLocalId": "<省略可。連投の続きなら、親の id>"
```

**時刻は、次のどれかにしてください。** 埋まっていない日の、埋まっていない枠から使います。

```
07:40  12:15  15:00  18:30  21:00  22:45
```

## 5. ★ 検査を通す（ここが関門）

```
node auto/verify-x.mjs
```

- **通ったら** → 6へ
- **落ちたら** → 落ちた投稿を直して、通るまで繰り返す。**直せなければ、その投稿を削除する。**
  **落ちた投稿を無理やり通すために、検査のほうを緩めないこと。**

検査が止めるもの: 薬機法の禁止表現 / 煽り（嘘・詐欺・騙し）/ 言い過ぎ（効かない・やめるべき）/ 文字数超過。

**なお「『効かない』とは書けません」のように、言い過ぎを断っている文は通ります。**
関門は「言い切っているか」を見ています。
