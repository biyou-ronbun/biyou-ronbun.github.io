### なぜ死ぬのか

> **Consumer Reports は1936年から広告を断り続けた消費者団体です。**
> その CR が、いま「CR Recommended」の認証マークの掲示料を、**評価した企業から取っています。**
>
> **これは「良い団体が堕落した話」ではありません。**
> **「判定を持つ者は、最後に判定を売る」という、構造そのものの話です。**
>
> **判定は資産になる。資産は換金圧力を持つ。理念の強度では止まらない。86年でも止まらなかった。**

**うちがスコア・ランキング・認証を1つも持たない理由は、意志ではなく構造です。**
**持たなければ、売る対象が存在しません。**

## 手順

### 1. 見る

読めるものだけを見てください。**開けなかったら「開けなかった」と書いてください。**

| メディア | 何を見るか |
|---|---|
| @cosme（https://www.cosme.net/ ） | 構造。何で流入を取っているか |
| LIPS（https://lipscosme.com/ ） | 悩みからの入口の作り方 |
| MimiTV（https://mimitv.co.jp/ ） | 記事の型 |
| 美的（https://www.biteki.com/ ） | 見出しの付け方 |
| VOCE（https://i-voce.jp/ ） | 同上 |

**海外も見てください。うちに近い立場のもの:**

| | |
|---|---|
| Lab Muffin Beauty Science | 化学者が書く。商品を勧めるが、根拠を示す |
| The Beauty Brains | 化粧品化学者 |
| Paula's Choice の Ingredient Dictionary | **★ 商品を売りながら成分辞典を持つ。うちが持てない形** |

### 2. 分ける

見つけたものを、**必ず2つに分けてください。**

**A. うちが取り込めるもの**
- **判定を含まない**（順位・点数・おすすめ・認証を作らない）
- **論文から導ける**
- **読者の言葉で書ける**

**B. うちが取り込めないもの ← ★ こちらが本体**
- **なぜ取り込めないかを、必ず書く**
- 「他が全部やっているのに、うちがやらない理由」を、毎週書き直す

### 3. 直す

**★ サイトを直すのは、1週間に1つだけです。**

たくさん直すと、**何が効いたのか分からなくなります。**
そして「たくさん直した」こと自体が目的になります。

**★ 「今週は何もしませんでした」は、正しい仕事です。**

**取り込めるものが1つも無かった週は、そう書いて終わってください。**
**無理に何かを見つけようとした瞬間、あなたはランキングを作り始めます。**

### 4. 記録する

`benchmark/media-playbook.md` に、次の形で**追記**してください。

```markdown
## 2026-XX-XX

### 開けたもの / 開けなかったもの
（開けなかったものは、必ず書く。開けたふりをしない）

### A. 取り込めるもの
（無ければ「無し」）

### B. ★ 取り込めないもの ← 本体
| 見つけた型 | どこで | なぜ取り込めないか |
|---|---|---|
| ランキング | @cosme | 判定は資産になり、最後に売られる（CR、86年） |

### 今週直したもの
（1つだけ。無ければ「何もしませんでした」）
```
